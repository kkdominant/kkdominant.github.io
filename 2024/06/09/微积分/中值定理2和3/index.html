中值定理2和3

发表于2024-06-09|更新于2024-06-18|微积分
|阅读量:

其中第二个中值定理有些难度、第一个是很经典的题目。

  1. 设$f\left( x\right)$ 在$\left\lbrack{0,1}\right\rbrack$上连续,在$\left({0,1}\right)$内可导,且$f\left( 0\right) = 0,f\left( 1\right) = 1$,试证对任意 正数${a}{1},{a}{2},\cdots ,{a}{n}$,在$\left( {0,1}\right)$内一定存在互不相同的$n$个数,$0 < {\xi }{1} < {\xi }{2} < \cdots < {\xi }{n} < 1$使得:

$$\frac{ {a}{1} }{ {f}^{\prime }\left( {\xi }{1}\right) } + \frac{ {a}{2} }{ {f}^{\prime }\left( {\xi }{2}\right) } + \cdots + \frac{ {a}{n} }{ {f}^{\prime }\left( {\xi }{n}\right) } = {a}{1} + {a}{2} + \cdots + {a}_{n}.$$

  1. 设$f\left( x\right)$在$\left\lbrack {0,a}\right\rbrack$上可导$\left( {a > 0}\right) ,f\left( 0\right) = 1,f\left( a\right) = 0$,证明; 在$\left( {0,a}\right)$上存在${x}{1} < {x}{2}$使得；

$$
{f}^{\prime }\left( {x}{1}\right) {f}^{\prime }\left( {x}{2}\right) = \frac{1}{ {a}^{2} }
$$

note：这两题实际上都是用的介值定理+Lagrange中值定理来证明的

proof

$$\frac{ {a}{1} }{ {f}^{\prime }\left( {\xi }{1}\right) } + \frac{ {a}{2} }{ {f}^{\prime }\left( {\xi }{2}\right) } + \cdots + \frac{ {a}{n} }{ {f}^{\prime }\left( {\xi }{n}\right) } = {a}{1} + {a}{2} + \cdots + {a}{n}.$$
等价于证明
$$
\frac{a_1}{\left( a_1+a_2+\cdots +a_n \right) f^{\prime}\left( \xi 1 \right)}+\frac{a_2}{\left( a_1+a_2+\cdots +a_n \right) f^{\prime}\left( \xi 2 \right)}+\cdots +\frac{a_n}{\left( a_1+a_2+\cdots +a_n \right) f^{\prime}\left( \xi n \right)}=1
$$
即
$$
\sum{i=1}^n{\frac{a_i}{f^{\prime}\left( \xi i \right) \sum{i=1}^n{a_i} }}=1
$$
由$f\left( x\right)$在$\left\lbrack {0,1}\right\rbrack$上连续,在$\left( {0,1}\right)$内可导,且$f\left( 0\right) = 0,f\left( 1\right) = 1$可由介值定理知:
存在$f(x_1)=\dfrac{a_1}{\left( a_1+a_2+\cdots +a_n \right)},
f(x_2)=\dfrac{a_1+a_2}{\left( a_1+a_2+\cdots +a_n \right)},\ldots, f(x_n)=1$即$f(x_i)=\dfrac{\sum_1^n a_i}{\left( a_1+a_2+\cdots +a_n \right)}$(注意这里这两个端点值$x_n=1,x_0=0$)
则由Lagrange中值定理知:
存在$\xi_i \in (x{i-1},x_i)$，使得:
$$
f^{\prime}\left( \xi i \right) =\frac{f\left( x_i \right) -f\left( x{i-1} \right)}{x_i-x{i-1} }=\frac{a_i}{\left( x_i-x{i-1} \right) \left( a_1+a_2+\cdots +a_n \right)}
$$

显然有$0 < {\xi }{1} < {\xi }{2} < \cdots < {\xi }{n} <1$,于是:
$$
\frac{a_i}{f^{\prime}\left( \xi i \right) \sum{i=1}^n{a_i} }=x_i-x{i-1}
$$
故
$$
\sum_{i=1}^n{\frac{a_i}{f^{\prime}\left( \xi i \right) \sum{i=1}^n{a_i} }}=\sum_{i=1}^n{\left( x_i-x_{i-1} \right)}=x_n-x_0=1
$$

这一题有些难度、中介点不好找。

$f\left( x\right)$在$\left\lbrack {0,a}\right\rbrack$上可导$\left( {a > 0}\right) ,f\left( 0\right) = 1,f\left( a\right) = 0$,令$g(x)=f(x)-\dfrac{x}{a}$,则$g(0)>0,g(a)<0$,则由介值定理知存在点$c \in(0,a)$使得,$g(c)=f(c)-\dfrac{c}{a} =0$,在$(0,\dfrac{c}{a}),(\dfrac{c}{a},1)$上分别运用Lagrange中值定理可得:
$$
f^{\prime}\left( \xi _1 \right) =\frac{f\left( c \right) -f\left( 0 \right)}{c}=\frac{c-a}{ac}
\
f^{\prime}\left( \xi _2 \right) =\frac{f\left( a \right) -f\left( c \right)}{a-c}=\frac{-c}{a\left( a-c \right)}
$$
因此:
$$
f^{\prime}\left( \xi _1 \right) f^{\prime}\left( \xi _2 \right) =\frac{c-a}{ac}\frac{-c}{a\left( a-c \right)}=\frac{1}{a^2}
$$

可以看出可以关键一步就是构造辅助函数$g(x)=f(x)-\dfrac{x}{a}$，这是怎么想到的呢?就是先随便试一个点c、然后分别运用Lagrange中值定理、看看什么时候成立，然后就是找到特殊点
