伽马分布与泊松分布的联系

发表于2024-06-18|更新于2024-06-18|统计学
|阅读量:

电子产品的失效常常是由子外界的 “冲击引起”. 若在 $\left( {0,t}\right)$ 内发生冲击的次数 $N\left( t\right)$ 服从 参数为 ${\lambda t}$ 的泊松分布,试证第 $n$ 次冲击来到的时间 ${S}_{n}$ 服从伽玛分布 ${Ga}\left( {n,\lambda }\right)$ .

主要要证明的就是(2)式

$$
\small
\begin{align*}
&\left( 1 \right)\int {x}^{+\infty }\frac{1}{\Gamma \left( \alpha \right) }{z}^{\alpha - 1}{\mathrm{e} }^{-z}\mathrm{;d}z = \sum \limits{ {y = 0} }^{ {\alpha - 1} }\frac{ {x}^{y}{\mathrm{e} }^{-x} }{y!},;\alpha = 1,2,3,\cdots \
&\left( 2 \right) \sum_{k=n}^{\infty}{\frac{\left( \lambda t \right) ^k}{k!}e^{-\lambda t} }=\frac{\lambda ^n}{\varGamma \left( n \right)}\displaystyle\int_0^t{x^{n-1}e^{-\lambda x} }dx
,\quad\lambda>0 &\
&\left( 3 \right) \sum_{k=0}^{n-1}{\frac{\left( \lambda t \right) ^k}{k!}e^{-\lambda t} }=\frac{\lambda ^n}{\varGamma \left( n \right)}\displaystyle\int_t^{\infty}{x^{n-1}e^{-\lambda x} }dx,\quad\lambda > 0
&
\end{align*}
$$
提示：利用分部积分法）并将上式表示成 Poisson 随机变量与伽玛随机变量之 间的概率关系式.

可以看出这三题可以归纳为一题，这里介绍不同的方法：分布积分法证明第一题、用函数的方法证明第二题、第三题是第二题的简单变形

第一题:

proof：

记$S\left( \alpha \right) =\displaystyle\int_x^{+\infty}{\dfrac{1}{\Gamma \left( \alpha \right)} }z^{\alpha -1}\mathrm{e}^{-z};\mathrm{d}z,\alpha =1,2,3,\cdots$

易得$S\left( 1 \right) =e^{-x}=\dfrac{x^0e^{-x} }{1!}$ ,
则由分部积分法可得:
$$
\begin{align*}
S\left( \alpha \right) &=\displaystyle\int_x^{+\infty}{\frac{1}{\Gamma \left( \alpha \right)} }z^{\alpha -1}\mathrm{e}^{-z};\mathrm{d}z
&\
&=-\displaystyle\int_x^{+\infty}{\frac{1}{\Gamma \left( \alpha \right)} }z^{\alpha -1};\mathrm{de}^{-z}
&\
&=\frac{-\mathrm{e}^{-z}z^{\alpha -1} }{\Gamma \left( \alpha \right)}\mid_{x}^{+\infty}+\displaystyle\int_x^{+\infty}{\frac{1}{\Gamma \left( \alpha -1 \right)} }z^{\alpha -2}\mathrm{e}^{-z};\mathrm{d}z
&\
&=\frac{\mathrm{e}^{-x}x^{\alpha -1} }{\Gamma \left( \alpha \right)}+S\left( \alpha -1 \right)
&
\end{align*}

$$
由$S(\alpha)$递推式可得:
$$
\begin{align*}
S\left( \alpha \right) &=\sum_{i=2}^{\alpha}{\left[ S\left( i \right) -S\left( i-1 \right) \right]}+S\left( 1 \right)
&\
&,, =\sum_{i=2}^{\alpha}{\left( \frac{\mathrm{e}^{-x}x^{i-1} }{\Gamma \left( i \right)} \right)}+\frac{x^0e^{-x} }{1!}
&\
&,, =\sum_{i=1}^{\alpha}{\left( \frac{\mathrm{e}^{-x}x^{i-1} }{\Gamma \left( i \right)} \right)}
&\
&,, \xrightarrow{y=i-1}\sum_{y=0}^{\alpha -1}{\left( \frac{\mathrm{e}^{-x}x^y}{\Gamma \left( y+1 \right)} \right)}=\sum_{y=0}^{\alpha -1}{\frac{x^y\mathrm{e}^{-x} }{y!} }
&
\end{align*}

$$
证毕

第二题和第三题：

分析:
如果$f^{\prime}\left( x \right) -g^{\prime}\left( x \right) =0$,且$f\left( 0 \right) =g\left( 0 \right)$ 显然有$f\left( x \right) =g\left( x \right)$

令$f\left( t \right) =\displaystyle\sum_{k=n}^{\infty}{\frac{\left( \lambda t \right) ^k}{k!}e^{-\lambda t} },f\left( 0 \right) =0$
$$

\begin{align*}

&f^{\prime}\left( t \right) =\sum_{k=n}^{\infty}{\frac{\lambda ^kkt^{k-1} }{k!}e^{-\lambda t} }-\lambda \sum_{k=n}^{\infty}{\frac{\left( \lambda t \right) ^k}{k!}e^{-\lambda t} }
&\
&=e^{-\lambda t}\left[ \sum_{k=n}^{\infty}{\frac{\lambda ^kt^{k-1} }{\left( k-1 \right) !}-\sum_{k=n}^{\infty}{\frac{\lambda ^{k+1}t^k}{k!} }} \right]
&\
&=e^{-\lambda t}\left[ \sum_{k=n}^{\infty}{\frac{\lambda ^kt^{k-1} }{\left( k-1 \right) !}-\sum_{k=n+1}^{\infty}{\frac{\lambda ^kt^{k-1} }{\left( k-1 \right) !} }} \right]
&\
&=\frac{e^{-\lambda t}\lambda ^nt^{n-1} }{\left( n-1 \right) !}
&\
\end{align*}
$$
令$g\left( t \right) =\dfrac{\lambda ^n}{\varGamma \left( n \right)}\displaystyle\int_0^t{x^{n-1}e^{-\lambda x} }dx,g\left( 0 \right) =0$

则有:
$
{g}^{\prime }\left( t\right) = \dfrac{ {\lambda }^{n} }{\Gamma \left( n\right) }{t}^{n - 1}{e}^{ - {\lambda t} } = \dfrac{ {e}^{ - {\lambda t} }{\lambda }^{n}{t}^{n - 1} }{\left( {n - 1}\right) !} = {f}^{\prime }\left( t\right)
$

$
\because {f}^{\prime }\left( t\right) - {g}^{\prime }\left( t\right) = 0
$

说明 $f\left( t\right) - g\left( t\right) = C,C$ 为常数

又 $\because f\left( 0\right) - g\left( 0\right) = 0$

可得 $f\left( t\right) = g\left( t\right)$

即 $\mathop{\sum }\limits_{ {k = n} }^{\infty }\dfrac{ {\left( \lambda t\right) }^{k} }{k!}{e}^{-{\lambda t} } = \dfrac{ {\lambda }^{n} }{\Gamma \left( n\right) }{\displaystyle\int }_{0}^{t}{x}^{n - 1}{e}^{-{\lambda x} }{dx}$

第三题是第二题的简单变形

$$
\begin{align*}
&\sum_{k=0}^{n-1}{\frac{\left( \lambda t \right) ^k}{k!}e^{-\lambda t} }
&\
&=\sum_{k=0}^{\infty}{\frac{\left( \lambda t \right) ^k}{k!}e^{-\lambda t} }-\sum_{k=n}^{\infty}{\frac{\left( \lambda t \right) ^k}{k!}e^{-\lambda t} }
&\
&=1-\sum_{k=n}^{\infty}{\frac{\left( \lambda t \right) ^k}{k!}e^{-\lambda t} }
&\
&\frac{\lambda ^n}{\varGamma \left( n \right)}\displaystyle\int_t^{\infty}{x^{n-1}e^{-\lambda x} }dx
&\
&=\frac{\lambda ^n}{\varGamma \left( n \right)}\displaystyle\int_0^{\infty}{x^{n-1}e^{-\lambda x} }dx-\frac{\lambda ^n}{\varGamma \left( n \right)}\displaystyle\int_0^t{x^{n-1}e^{-\lambda x} }dx
&\
&=1-\frac{\lambda ^n}{\varGamma \left( n \right)}\displaystyle\int_0^t{x^{n-1}e^{-\lambda x} }dx

\end{align*}
$$

由(2)知$\displaystyle\sum_{k=n}^{\infty}{\frac{\left( \lambda t \right) ^k}{k!}e^{-\lambda t} }=\frac{\lambda ^n}{\varGamma \left( n \right)}\displaystyle\int_0^t{x^{n-1}e^{-\lambda x} }dx$

故$\displaystyle\sum_{k=0}^{n-1}{\frac{\left( \lambda t \right) ^k}{k!}e^{-\lambda t} }=\frac{\lambda ^n}{\varGamma \left( n \right)}\displaystyle\int_t^{\infty}{x^{n-1}e^{-\lambda x} }dx$

解决了这个恒等式下面这题迎刃而解了
