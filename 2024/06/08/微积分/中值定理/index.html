中值定理1

一个中值定理题目

设 $f\left( x\right)$ 在 $ \left\lbrack {0,1} \right\rbrack $上二阶可导,且 $f\left( 0\right) = {f} ^{\prime } \left( 0\right) = {f} ^{\prime \prime } \left( 0\right) = f\left( 1\right) = 0 $ ,证明存在 $ \xi \in (0,1)$ 使得：

$$
{\xi } ^{2} {f} ^{\prime \prime } \left( \xi \right) - {2f} \left( \xi \right) = 0
$$

这题我感觉证明起来有些难度、有些步骤不容易想到、先给出答案、最后讲一下我的思路

证明

设$G\left( x \right) =\begin{cases}
\dfrac{f\left( x \right)} {x^2} , x\in \left( 0,1 \right]\
0, x=0\
\end{cases} $ ,易验证$G(x)$ 在[0,1]上连续可导
当$x\ne 0$时:
$$
G^{\prime} \left( x \right) =\frac{f^{\prime} \left( x \right) x^2-2xf\left( x \right)} {x^4} =\frac{f^{\prime} \left( x \right) -\dfrac{2} {x} f\left( x \right)} {x^2}
$$

$G(0)=G(1)=0$由罗尔定理可知、存在$\xi_{1} \in (0,1)$ 使得 $G^{\prime} (x) =0$即：

$$
f^{\prime} \left( \xi _1 \right) -\frac{2} { {\xi _1} } f\left( \xi _1 \right) =0\Rightarrow {\xi _1} ^2f^{\prime} \left( \xi _1 \right) -2\xi _1f\left( \xi _1 \right) =0
$$
设$H\left( x \right) =x^2f^{\prime} \left( x \right) -2xf\left( x \right)$ ,则$H(\xi_1)=H(0)=0$ 则由罗尔定理知:
存在$\xi_2\in(0,1)$使得$ H^{\prime} (x)^ =0$即:
$$
{\xi _2} ^2f^{\prime\prime} \left( \xi _2 \right) -2f\left( \xi _2 \right) =0
$$
证毕

思路

$$
\begin{align*}
&x^2f^{‘’} \left( x \right) -2f\left( x \right) =0
&\
&\Rightarrow x^2f^{‘’} \left( x \right) +2x^{\prime} f\left( x \right) -2x^{\prime} f\left( x \right) -2f\left( x \right) =0
&\
&\Rightarrow \left( x^2f^{\prime} \left( x \right) \right) ^{\prime} -2\left( xf\left( x \right) \right) ^{\prime} =0
&
\end{align*}
$$
设$H\left( x \right) =x^2f^{\prime} \left( x \right) -2xf\left( x \right)$ ,$H(0) =0$再找一个零点使用罗尔定理即可
$$x^2f^{\prime} \left( x \right) -2xf\left( x \right) =0\Rightarrow f^{\prime} \left( x \right) -\frac{2} {x} f\left( x \right) =0$$
构造$G\left( x \right) =f\left( x \right) e^{\int{-\frac{2} {x} dx} } =\frac{f\left( x \right)} {x^2} ,G(1)=0$,但是这个$ G(x) $ 在$ 0 $ 处不连续、无法使用题目所给的条件
因此构造:
$$G\left( x \right) =\begin{cases} \dfrac{f\left( x \right)} {x^2} , x\in \left( 0,1 \right]\0, x=0\
\end{cases}
$$

容易验证$G(0) =G(1)=0$且一阶连续可导、因此可以使用罗尔定理即$ G^{\prime} \left( \xi _1 \right) =0 $即$H\left( 0 \right) =H\left( \xi _1 \right) =0\Rightarrow H\left( \xi _2 \right) =0$
