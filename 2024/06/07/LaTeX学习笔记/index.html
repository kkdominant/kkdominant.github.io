\section{概率和统计}

\begin{block}{变量}<0->
	变量是能取多个值的任意属性或符号、变量$ X $ 所取值$ x $ 的概率记作$ P(X=x) $;变量可分为连续变量和离散变量

\begin{block}{事件}<0->
	为一个变量或者一个变量集合指定一个值(或一组值)成为一个事件。

	从概率的角度考虑事件的方法是:任何陈述性说明(说明是可能是真或假)都是一个事件
\end{block}
\end{frame}

beamer修改超链接颜色报错

使用 \usepackage[colorlinks=true, urlcolor=blue]{hyperref} % 只将超链接设置为蓝色设置 url颜色时、会报错

这是因为在使用 LaTeX 的 Beamer 类创建演示文稿时， Beamer 自身已经自动加载了 hyperref 宏包，并对其进行了一些特定的配置。因此，当你尝试通过 \usepackage{hyperref} 命令来显式加载或配置 hyperref 时，可能会与 Beamer 的内部设置冲突，导致编译错误。

如果你取消\usepackage{hyperref} 发现超链接还可以使用

正确的配置应该是

\hypersetup{
  colorlinks=true,
  urlcolor=blue,
  linkcolor=blue,     % 内部链接颜色
  citecolor=blue     % 引用链接颜色
}

公式显示问题

在align环境报错显示缺少$$

\begin{align*}

	P\left( y\mid do\left( x \right) \right) &=\sum_z{P\left( Y=y\mid X=x,Z=z \right) P\left( Z=z \right)}\,\, \left( \text{校正公式} \right) \\
	&=\sum_z{\frac{P\left( Y=y,X=x,Z=z \right)}{P\left( X=x,Z=z \right) /P\left( Z=z \right)}}\left( \text{条件概率} \right) \\
	&\,\,             =\sum_z{\frac{P\left( Y=y,X=x,Z=z \right)}{P\left( X=x\mid Z=z \right)}}

\end{align*}
````
上面的公式渲染失败、找了一下原因、发现是空行引起的错误
修改为:

```latex
\begin{align*}
	P\left( y\mid do\left( x \right) \right) &=\sum_z{P\left( Y=y\mid X=x,Z=z \right) P\left( Z=z \right)}\,\, \left( \text{校正公式} \right) \\
	&=\sum_z{\frac{P\left( Y=y,X=x,Z=z \right)}{P\left( X=x,Z=z \right) /P\left( Z=z \right)}}\left( \text{条件概率} \right) \\
	&\,\,             =\sum_z{\frac{P\left( Y=y,X=x,Z=z \right)}{P\left( X=x\mid Z=z \right)}}
\end{align*}

```
