快捷键shortcuts

基本命令

  1. sysuse auto 导入数据"auto"： (1978 automobile data)

  2. br=browse 浏览导入的数据

  3. des=describe 描述数据

  4. list make 列举data的make列 (制造商)

  5. notes:查看数据来源

  1. sum=summarize 对所有变量分析

    (字符串类型不进行统计分析eg:make) obs就是observation应该就是观察的数量

    rep78的obs=69表明这个变量有5个缺失值

    summarize price : 描述具体的某个变量

    summarize price,detail, : 描述价格在每个百分位数上的数值

    summarize mgp if foreign==1先筛选foreign=1的变量然后再分析mpg

    这里示例数据中foreign只取两个值0,1可以写两行代码summarize mgp if foreign==1和summarize mgp if foreign==0但是如果类别比较多怎么办呢、比如rep78的取值是1,2,3,4,5还能写5行代码吗?stata中有如下命令:

    注意这里不止5类、因为他把缺失值也算做了一类

  2. codebook rep78查看具体的变量:可以查看变量值范围、缺失值个数、数据类型

  3. br if missing(rep78):查看某个变量具体的缺失值

    这个命令等价于br if rep78==.注意有两个等于号

    也可以list rep78 if rep78==.

  4. drop if missing(rep78)删除rep78的缺失值

  5. tabulate rep78查看变量取值的频数。通过上面的codebook rep78我们知道rep78是整数类型、取值范围是[1,5]这个命令就是查看取每个数值的频数

  6. tabulate rep78 foreign现在我们知道rep79每个数值的频数了、现在我们想更近一步的知道每个数值在各个国家的分布

  7. drop lengh删除变量length

  8. reg y x1 x2 回归

  9. cls 清屏

  10. db regress 查找回归的界面在哪

统计模型及检验

相关系数

  1. correlate x y 计算x和y的相关性、

    只需要看左下角的数据-0.7859表示这两个变量的相关系数是-0.7958，表格中的1是自己和自己的相关系数。

  2. by z ，sort :correlate x y 在z变量下将数据分类、在每个类别下查看x和y 的相关系数

    eg:by foreign ,sort: correlate mpg length分别查看本地中mpg和length的correlation和外国mpg和length的coorrelation

  3. correlate x1 x2 x3 x4 x5多个变量之间的相关系数

    eg: correlate price mpg length weight

    同样我们只需要关注主对角线左侧的数值即可

两变量t检验

ttest mpg ,by(foreign)检验不同国家的mpg是否具有显著差异
